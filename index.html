Сгораемые бонусы

Особенностью платформы iProBonus является наличие движка сгораемых бонусов.

Можно создать любое количество типов бонусов для которых можно указать:

  • Название
  • Срок сгорания бонусов

При начислении бонусов система автоматически отслеживает момент времени, когда они должны сгореть.
На главном экране клиенту показывается текущее количество бонусов, которые ему доступны сейчас.
Также отражается количество бонусов которые сгорит в ближайший период времени и дату, когда они сгорят.
На отдельном экране отображается доступное количество бонусов по типам.
Клиенту доступна вся история движения бонусов на отдельном экране.
Бонусы могут быть начислены в автоматическом режиме (см триггерное начисление бонусов),
при осуществлении покупки (если не было оплаты бонусами) в соответствии с установленными правилами программы лояльности, акционно
(см, массовое начисление бонусов).

Промокоды

На главном экране приложения выводится модуль, который позволяет перейти на экран применения промокода. Промокод можно ввести вручную, а также можно считать в виде QR кода. После успешного применения промокода у клиента появляется та сумма баллов, которая прописана в условиях применения промокодов.

Особенностью применения промокодов на платформе является то, что они не дают прямой скидки при их применении. По промокодам можно получить сгораемые бонусы. Это позволяет вовлекать в программу лояльности не только клиентов, которые готовы сделать покупку прямо сейчас, а всех потенциальных клиентов, которые не готовы купить сейчас. Мы не призываем человека сразу потратить деньги, а предлагаем зарегистрироваться программе лояльности, чтобы применить промокод. Это дает возможность продолжить общение с клиентом и стимулировать его сделать покупки в будущем. Такое поведение выгодно отличает промокоды платформы iProBonus от промокодов других платформ, т.к. позволяет вовлечь в программу лояльности намного больше человек.

Платформа поддерживает следующие типы промокодов:

  • Массовый Одноразовый промокод - после его применения, невозможно применить его еще раз - выдается предупреждение, что промокод погашен. Обычно такой тип промокод представлен в виде QR кода.
  • Общий промокод для однократного применения - каждый клиент может применить данный промокод для себя один раз. При попытке повторного применения будет выдаваться сообщение, что клиент уже применил данный промокод. Данный тип промокода может быть применен любым количеством клиентов. Пример такого типа промокодов Mart82021 - с начислением 500 бонусов, которые будут доступны в течении 10 дней с момента активации промокод.
  • Общий промокод лимитированный количеством применений. Может использоваться для создания эффекта лимитированного предложения.

Промокоды заводятся в админ панели.

При заведении промокод выбирается его тип. Если это тип Массовый одноразовый промокод, то предлагается либо сгенерировать необходимое число промокодов, либо загрузить его из внешнего источника (txt файла, каждый промокод разделен символом перевода строки). Сгенерированные промокоды можно в любой момент времени выгрузить в файл формата csv. Возможен сценарий, когда производится акция с внешними игроками по кросс применению промокодов. В этом случае создается два типа промокода: собственные промокоды и промокоды внешнего бизнеса. В первом случае сгенерированные промокоды выгружаются и передаются внешнему партнеру. Для второго случая производится загрузка переданных от партнера по акции промокодов. Контроль применения промокодов в обоих случаях ведется на стороне платформы iProBonus. Для каждого созданного типа промокодов необходимо указать следующие параметры:

  • Внутреннее название промокода
  • Название и описание для клиента
  • Тип начисляемых бонусов
  • Сумму начисляемых бонусов
  • Период действия промокода.

Важно! Период действия промокода не будет совпадать с периодом сгорания бонусов. Это независимые параметры. Т.е. если мы выпускаем промокод с периодом действия с 1 января по 10 января и устанавливаем тип бонусов, сгорания которых происходит через 7 дней, то период действия промомокода может завершится, но начисленные бонусы еще будут действовать. Так если клиент использует промокод 9 января, ему будут начислены бонусы, которые сгорят 16 января. При этом 11 января он уже не сможет активировать промокод.
